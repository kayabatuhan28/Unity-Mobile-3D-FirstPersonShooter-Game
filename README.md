# Unity-Mobile-3D-FirstPersonShooter-Game

Gameplay Video Link : https://www.youtube.com/watch?v=gbobXGkZBas&list=PLeUQ9Opn9i7PpZXLhEPG_rDb0LjjYapMA&index=2

------------------------------------------------------------ TR ------------------------------------------------------------
- Mobil için birinci şahıs nişancı (FPS) oynanış
- Dokunmatik tabanlı nişan alma ve ateş etme sistemi
- Silah ateşleme ve yeniden doldurma (reload) mekanikleri
- Mermi, cephane yönetim sistemi
- Yerden cephane toplama (ammo pickup) sistemi
- Silah geliştirme sistemi (hasar, mermi kapasitesi vb. artırma)
- Farklı silah kullanım seçenekleri
- El bombası atma sistemi ve patlama hasarı
- Devriye, takip ve saldırı davranışlarına sahip düşman yapay zekâsı
- Düşmanların silah ve patlama seslerine tepki vermesi
- Oyuncu ve düşman sağlık sistemi
- Hasar geri bildirimi ve isabet tespiti (hit detection)
- Dosyadan veri okuma (ayarlar,ilerleme)
- Veri kaydetme sistemi (oyuncu ilerlemesi, mermi sayısı, yükseltmesi vb.)
- Mobil için optimize edilmiş kontroller ve performans
- Temel kullanıcı arayüzü (can barı, mermi sayacı)
- Temel FPS mekaniklerini sergileyen oynanış odaklı bir prototip

------------------------------------------------------------ EN ------------------------------------------------------------
- Mobile first person shooter (FPS) gameplay
- Touch based aiming and shooting system
- Weapon firing and reload mechanics
- Ammo and ammunition management system
- Ammo pickup system (collecting ammo from the ground)
- Weapon upgrade system (increasing damage, ammo capacity, etc.)
- Multiple weapon usage options
- Grenade throwing system with explosion damage
- Enemy AI with patrol, chase, and attack behaviors
- Enemies reacting to gunshot and explosion sounds
- Player and enemy health system
- Damage feedback and hit detection
- Data loading from file (settings, progress)
- Data saving system (player progress, ammo count, upgrades, etc.)
- Mobile optimized controls and performance
- Basic user interface (health bar, ammo counter)
- Gameplay focused prototype showcasing core FPS mechanics
